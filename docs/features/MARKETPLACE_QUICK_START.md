## üîß Troubleshooting

### "No apps found"
- Refresh the page
- Check your internet connection
- Verify you're logged in with correct role

### Can't install an app
- Check if you have the required role (admin apps need admin access)
- Ensure you're not at max installed apps limit
- Try logging out and back in

### App not working
- Check if the app requires additional setup
- Verify external connections are configured (Admin only)
- Clear browser cache and reload
