‚ùå Broken Approach (PR #92)

‚úÖ Fixed Approach (PR #93)

üìö Key Lesson

When an element has position: fixed, it normally positions relative to the viewport. However, if any ancestor element has position: fixed, transform, perspective, filter, or certain other properties, the fixed element positions relative to that ancestor instead of the viewport. This is what broke the modal in PR #92.
